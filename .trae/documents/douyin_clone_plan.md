# 轻量级抖音Web视频播放器产品规划

## 1. 项目概述

基于提供的API，开发一个轻量级的抖音Web视频播放器，专注于核心功能，用于个人娱乐使用。支持随机视频播放、视频分类选择和基本的视频控制功能。

## 2. API分析

### 2.1 API基本信息
- API地址：https://wzapi.com/api/sjxjsp
- 请求方式：GET
- 返回格式：JSON

### 2.2 请求参数
| 参数名 | 必填 | 类型 | 说明 |
|-------|------|------|------|
| format | 是 | string | 输出格式，固定为json |
| category | 是 | string | 视频分类，可选值：baisi、heisi、gaozhiliang、chuanda、shejie、qingchun、rewu、nvda、jk |

### 2.3 返回参数
| 参数名 | 类型 | 说明 |
|-------|------|------|
| timestamp | string | 时间戳 |
| video | string | 视频链接 |
| category | string | 当前视频分类 |
| id | string | 播放视频ID |
| status | string | 返回状态 |

## 3. 技术选型

### 3.1 前端技术栈
- **框架**：React + TypeScript
- **构建工具**：Vite
- **样式方案**：Tailwind CSS
- **视频播放**：HTML5 Video API
- **HTTP客户端**：Axios
- **状态管理**：React useState/useEffect (轻量级)

### 3.2 项目结构
```
/src
  /components
    /VideoPlayer
    /CategorySelector
    /Loading
  /utils
    /api.ts
  App.tsx
  main.tsx
```
